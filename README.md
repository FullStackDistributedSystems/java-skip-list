# Lock-free Skip List

## Introduction

A Skip List is a data structure designed to

| Operation        | Array List       | Linked List      | Skip List          |
|------------------|------------------|------------------|--------------------|
| Access           | $\Theta(1)$      | $\Theta(n)$      | $\Theta(log(n))$   |
| Search           | $\Theta(n)$      | $\Theta(n)$      | $\Theta(log(n))$   |
| Insert           | $\Theta(n)$      | $\Theta(1)$      | $\Theta(log(n))$   |
| Remove           | $\Theta(n)$      | $\Theta(1)$      | $\Theta(log(n))$   |
| Space Complexity | $\Theta(n)$      | $\Theta(n)$      | $\Theta(n\:log(n))$ |
